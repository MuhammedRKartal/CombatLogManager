# CombatLogManager

CombatLogManager is a lightweight World of Warcraft (WotLK 3.3.5a) addon that automatically manages the combat log based on the raid instance you are in.

---

## Features

- Automatically enables combat logging when entering selected raid instances.
- Automatically disables combat logging when leaving raids or entering non-selected instances.
- Simple UI to choose which raids should trigger combat logging.
- Movable minimap button with visual status:
  - Green icon: combat logging enabled
  - Red icon: combat logging disabled
- Full sync with the `/combatlog` slash command.

---

## Supported Raids

- Icecrown Citadel  
- The Ruby Sanctum  
- Trial of the Crusader  
- Vault of Archavon  
- Naxxramas  
- Onyxia's Lair  
- Ulduar  
- The Obsidian Sanctum  

---

## Commands

- `/clm`  
  Open or close the CombatLogManager UI.
  
- `/combatlog`  
  Manual combat log toggling is detected and reflected in the UI and minimap icon.

---

## Minimap Button

- Left Click: Open / close the UI  
- Right Click: Toggle combat logging  
- Drag: Move the button around the minimap
